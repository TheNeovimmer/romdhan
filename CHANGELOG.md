# Changelog

All notable changes to this project will be documented in this file.

## [1.0.0] - 2026-02-19

### Added
- Initial release of Romdhan CLI
- Prayer times command with support for multiple calculation methods
- Ramadan countdown showing days until Ramadan or Eid
- Random hadith display with Arabic and English translations
- Quran reader with support for all 114 surahs
- Interactive Zakat calculator with multiple asset categories
- Digital tasbih counter for dhikr
- Settings management with auto-location detection
- Daily Ramadan info with moon phases and motivational quotes
- Full TypeScript support with proper typing
- ESLint and Prettier configuration
- Comprehensive README with examples and screenshots
